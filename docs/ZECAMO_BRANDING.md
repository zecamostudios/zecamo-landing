## Paleta de Colores

| Token | Variable CSS | Valor Hex | Uso |
|-------|-------------|-----------|-----|
| Primary | `--color-primary` | `#2B5BFF` | Azul eléctrico — botones CTA, links, highlights |
| Background base | `--color-bg-base` | `#0A0F1F` | Fondo principal (azul marino muy oscuro) |
| Background surface | `--color-bg-surface` | `#0D1526` | Cards, secciones con contraste suave |
| Background elevated | `--color-bg-elevated` | `#111B35` | Hover states, elementos elevados |
| Secondary | `--color-secondary` | `#0A1A4F` | Azul marino medio — fondos alternativos |
| Accent | `--color-accent` | `#4F7BFF` | Hover, focus, estados interactivos |
| Text primary | `--color-text-primary` | `#FFFFFF` | Texto principal |
| Text secondary | `--color-text-secondary` | `#93A4D1` | Texto de apoyo, descripciones |
| Text muted | `--color-text-muted` | `#5A6A8A` | Placeholders, etiquetas pequeñas |
| Border subtle | `--color-border-subtle` | `#1C2B50` | Bordes suaves |
| Border default | `--color-border-default` | `#243460` | Bordes estándar |

*Todas estas variables están definidas en `src/styles/variables.css`.*

## Tipografía

| Rol | Fuente | Variable CSS | Uso |
|-----|--------|-------------|-----|
| Body / sans | Geist | `--font-sans` | Texto corrido, UI |
| Display | Space Grotesk | `--font-display` | Títulos grandes, hero, section headers |
| Mono | Geist Mono | `--font-mono` | Código, datos técnicos |

**Escala tipográfica:** Definida en `src/styles/variables.css` con tokens `--text-display-xl` hasta `--text-label-sm`.

## Estilo Visual Objetivo

- **Modo oscuro por defecto** (dark mode only en la landing)
- **Minimalista y futurista**, pero con calidez profesional
- Gradientes sutiles azul eléctrico → azul oscuro
- Mucho espacio en blanco (airiness)
- Tipografía grande y bold en los títulos
- Sin clutter — cada elemento tiene propósito
